## Getting Started

### Prerequisites
- A web browser

### Installation
1. Clone the repository:
  ```bash
  git clone https://github.com/T-0n-1/www-interactivity-h2.git
```
2. Navigate to the project directory:
  ```bash
  cd www-interactivity-h2
```
3. Open the index.html file in your web browser.
